Example Interaction Flow

  • User creates profile in 'Music Tinder".
  • User connects their email and spotify to the app
  • App reviews the most played songs from a user and adds it to a "container" within the users profile.
  • App then tries to find other users with similar song choices
  • App shows recomendations of other users who liked similar artists/podcasts
  • User is able to swipe left (rejection) and right (match) when they see a user who they would like to meet.
  • App sends notification to user about the match (this may be an additional paid feature)
  • Other user swipes left and right on you. If they match then a notification is sent to both matched users to prompt them to connect.
  • A messaging function will be added to allow each matched user to communicate.
